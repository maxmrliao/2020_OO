### 非功能性需求： 
##### 反應時間：使用者拍攝照片後上傳到伺服器，伺服器進行辨識後回傳到眼鏡的時間。
##### 可靠度：網路不穩、拍攝照片對比度不高皆可能影響辨識結果。
##### 效能：照片的辨識只能一次辨識一張。
